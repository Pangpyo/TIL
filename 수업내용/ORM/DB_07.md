### 3. Queryset 메소드 `create` 를 활용해서  `Director` 테이블에 아래 데이터를 추가하는 코드를 작성하세요.

| name | debut | country |
| --- | --- | --- |
| 봉준호 | 1993-01-01 | KOR |
| 김한민 | 1999-01-01 | KOR |
| 최동훈 | 2004-01-01 | KOR |
| 이정재 | 2022-01-01 | KOR |
| 이경규 | 1992-01-01 | KOR |
| 한재림 | 2005-01-01 | KOR |
| Joseph Kosinski | 1999-01-01 | KOR |
| 김철수 | 2022-01-01 | KOR |

> 코드 작성
> 

```python
import datetime
directors_tables = [('봉준호', datetime.date(1993, 1, 1)), 
('김한민', datetime.date(1999, 1, 1)), 
('최동훈', datetime.date(2004, 1, 1)), 
('이정재', datetime.date(2022, 1, 1)), 
('이경규', datetime.date(1992, 1, 1)), 
('한재림', datetime.date(2005, 1, 1)), 
('Joseph Kosinski', datetime.date(1999, 1, 1)), 
('김철수', datetime.date(2022, 1, 1)), 
]
for n, d in directors_tables :
    director = Director()
    director.name = n
    director.debut = d
    director.country = 'KOR'
    director.save()
```

```
봉준호 1993-01-01 00:00:00 KOR
김한민 1999-01-01 00:00:00 KOR
```

> 

```python

```

### 13. 본인이 생각하는 혹은 찾은 `get` 과 `filter` 의 차이를 작성하세요.

```
get은 내가 찾는 조건에 해당하는 하나의 값만을 리턴 할 수 있다. 그렇기에 prime key를 찾는데 유용 할 것 같다.
filter는 변수 명 그대로 내가 찾는 조건에 해당하는 모든 값들을 필터로 거르듯이 리턴해준다. 그 값이 1개든 여러개든 리턴 해 주는 것을 알 수 있었다.
```
